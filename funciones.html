Funciones

Una función es un conjunto de instrucciones que se agrupan para realizar una tarea concreta. Una de sus mayores ventajas es que se pueden reutilizar fácilmente.

La estructura básica de una función JavaScript se puede dividir en tres partes:

  1. Nombre: el nombre de la función.
  2. Argumentos: las variables o constantes a utilizar.
  3. Sentencias: las instrucciones de la función (lo que queremos que haga).
  4. Declaraciones de funciones: (describir o definir la función).
  5. Llamadas a funciones: (cuando y como dar la orden de que se ejecuten).
